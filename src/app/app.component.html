This project is a pathology lab website. Users can book an appointment as well as cancel/ reschedule it.

A technician can view a list of all the appointments booked with him/ her & cancel an appointment.

The admin can perform CRUD operations on appointments as well as users.

The external API's used were Onsched & ApiMed

A user can also check for probable issues without logging in using the symptom checker
